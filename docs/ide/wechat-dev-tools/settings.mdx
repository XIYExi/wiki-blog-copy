title: Settings
---

## formatOnSave

Configure in `settings.json`:

```json
{
  "editor.formatOnSave": true
}
```

## Prettier Extension

Configure the default format tool (prettier) for `javascript` and `json` in `settings.json`:

```json
{
  "[javascript]": {
    "editor.defaultFormatter": "esbenp.prettier-vscode"
  },
  "[json]": {
    "editor.defaultFormatter": "esbenp.prettier-vscode"
  }
}
```
