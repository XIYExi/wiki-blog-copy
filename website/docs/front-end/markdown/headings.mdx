title: Headings

## Markdown Syntax

```markdown
# 一级标题

## 二级标题

### 三级标题

#### 四级标题

##### 五级标题

###### 六级标题
```

## HTML Syntax

```html
<h1>一级标题</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>
<h4>四级标题</h4>
<h5>五级标题</h5>
<h6>六级标题</h6>
```

## Best Practice

为了兼容性，需要在 # 与 标题名称之间添加一个空格。

<table>

    </tr>
  </thead>

    <tr>

      </td>
    </tr>
    <tr>
      <td>
        <code>## heading</code>
      </td>
      <td>
        <code>##heading</code>
      </td>
    </tr>
  </tbody>
</table>
