## Block-level elements

HTML block-level elements like `<div>`，`<table>`，`<pre>`， and `<p>`，please use <TH>blank lines</TH> before and after them to separate from other content. Try not to use `tabs` or `spaces` to indent HTML elements, otherwise it will affect the formatting.

We <TH>cannot use Markdown syntax</TH> inside block-level elements. `<p>italic and **bold**</p>` will not work。
