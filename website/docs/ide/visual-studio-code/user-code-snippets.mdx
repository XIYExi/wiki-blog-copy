## Insert Snippet

在命令面板中使用 `Insert Snippet` 来查看当前文件语言支持的代码片段列表（包括用户自定义代码片段）。

## 制表符补全

Visual Studio Code 也支持使用制表符补全提示的代码，在 **settings.json** 中添加如下配置启用该功能：

```json
{
  "editor.tabCompletion": "on"
}
```
