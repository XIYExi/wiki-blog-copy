## Git Config

config 配置具有三个级别，依次是 System、Global、Local，后面的底层配置会覆盖前面的配置。

### 查看系统 Config

```bash title="Bash"
git config --system --list
```

### 查看当前用户 Config

```bash title="Bash"
git config --global --list
```

### 查看当前仓库 Config

```bash title="Bash"
git config --local --list
```
