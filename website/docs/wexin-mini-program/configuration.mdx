## Global configuration

根目录下 **app.json** 对微信小程序进行全局配置，决定页面文件的路径、窗口表现、设置网络超时时间、设置多 tab 等。

## Page configuration

每一个小程序页面也可以使用同名 .json 文件来对本页面的窗口表现进行配置，<TextHighlight>页面中配置项会覆盖 **app.json** 的 **window** 中相同的配置项</TextHighlight>。
