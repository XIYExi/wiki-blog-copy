## 无法连接到 GitHub 服务器

在 Push 代码到 GitHub 时遇到错误：Failed to connect to github.com port 443:connection timed out。

由于我使用了代理，但 vscode 并未尊重我的全局代理设置。

### 配置 vscode proxy

打开 **settings.json** ，添加如下 proxy 配置代码（禁用 SSL ，代理）：

```json
{
  "http.proxyStrictSSL": false,
  "http.proxy": "http://xxx.xxx.xxx.xxx:port"
}
```

此处 **port** 为代理端口号，我的代理端口号是 7890，即：**http://127.0.0.1:7890** 。

### 设置 Git 代理

在根目录为项目配置 Git 代理（禁用 SSL ，代理）：

```git
git config http.sslVerify false
git config http.proxy http://127.0.0.1:7890
```

至此，可以成功 Push 代码到 GitHub 服务器。
