## Markdown Syntax

```markdown
# Heading level 1

## Heading level 2

### Heading level 3

#### Heading level 4

##### Heading level 5

###### Heading level 6
```

## HTML Syntax

```html
<h1>Heading level 1</h1>
<h2>Heading level 2</h2>
<h3>Heading level 3</h3>
<h4>Heading level 4</h4>
<h5>Heading level 5</h5>
<h6>Heading level 6</h6>
```

## Best Practice

For compatibility, we should add a <Shortcut>space</Shortcut> between number signs `(#)` and heading name.

<table>
  <thead>
    <tr>
      <th>✅</th>
      <th>❌</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>
        <code># heading</code>
      </td>
      <td>
        <code>#heading</code>
      </td>
    </tr>
    <tr>
      <td>
        <code>## heading</code>
      </td>
      <td>
        <code>##heading</code>
      </td>
    </tr>
  </tbody>
</table>
