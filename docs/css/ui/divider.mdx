---
title: 分隔线
---

微信小程序中不支持 **&lt;hr&gt;** 标签，需要使用 **CSS** 实现分隔线。

```html title="HTML"
<div class="line" />
```

```css title="CSS"
.line {
  width: 100%;
  height: 1px;
  background: blue;
  margin: 0 auto;
}
```

绘制出分隔线如下：

<div
  style={{ width: "100%", height: "1px", background: "blue", margin: "0 auto" }}
/>
