## Block-level elements

HTML block-level elements like `<div>`，`<table>`，`<pre>`， and `<p>`，please use <TextHighlight>blank lines</TextHighlight> before and after them to separate from other content. Try not to use `tabs` or `spaces` to indent HTML elements, otherwise it will affect the formatting.

We <TextHighlight>cannot use Markdown syntax</TextHighlight> inside block-level elements. `<p>italic and **bold**</p>` will not work。
